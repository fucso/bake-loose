# 参照ドキュメントテンプレート

以下のテンプレートを使用して references/ ドキュメントを作成する。

---

```markdown
# {{ドキュメントタイトル}}

{{1-2文の概要}}

---

## 目次

{{100行を超える場合のみ記載}}

1. [{{セクション1}}](#{{アンカー1}})
2. [{{セクション2}}](#{{アンカー2}})
3. ...

---

## {{セクション1}}

{{本文}}

## {{セクション2}}

{{本文}}

---

## 適用例

{{ルールやパターンの具体的な適用例。不要な場合は省略可}}
```

---

## 記載ルール

- 100行を超える場合は目次を冒頭に配置する
- 他の references/ ドキュメントを参照しない（1階層参照の原則）
- 各セクションは自己完結させる
- Claude が既知の一般知識は記載しない
