# スキル作成ガイド

## 概要

スキルは Claude Code の能力を拡張するための構造化された知識・ワークフローの集合体。
`.claude/skills/{skill-name}/` ディレクトリに配置し、SKILL.md をエントリポイントとする。

## スキルの2種別

| 種別 | 説明 |
|------|------|
| **コマンドワークフロー型** | 手順・品質チェック・スクリプトを含む実行可能なワークフロー |
| **知識・ルール提供型** | ベストプラクティスやガイドラインを提供 |

### 判断基準

| 質問 | コマンドワークフロー型 | 知識・ルール提供型 |
|------|----------------------|-------------------|
| 実行順序が重要か？ | はい — ステップの順序が成果に影響 | いいえ — 参照順序は自由 |
| 品質チェックフェーズがあるか？ | はい — 検証・フィードバックループ | いいえ — 適用時の判断に委ねる |
| scripts/ が必要か？ | 多い — 定型処理の自動化 | 少ない — 通常不要 |
| ユーザーとの対話ポイントは？ | 明確に定義 — ステップごとに主体を指定 | 最小限 — ガイドラインの参照のみ |
